# Binary-Tree-Path-Sum
Given a binary tree and a number ‘S’, find if the tree has a path from root-to-leaf such that the sum of all the node values of that path equals ‘S’.

                                                                      12

                                                              7                  1

                                                        9              10                 5
                                                  
                                                  Sum: 23
                                                  output: True
